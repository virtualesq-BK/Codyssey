## 1. 업무 과업 입력 템플릿

| 항목 | 내용 |
|------|------|
| **업무과업** | 메일 초안 작성 |
| **사용자** | 해외 계약 법률 리걸테크 SaaS 플랫폼 기업 직원 |
| **목적** | 신규 출시 알림, 무료 체험 요청, 데모 예약, 유료 전환 유도, 회신 |
| **대상** | 잠재 고객인 중소기업 대표, 스타트업 대표, 법무팀 직원, 총무팀 직원, 인사팀 직원 |
| **산업군** | 무역업 / 제조업 / IT / 법무법인 / 스타트업 등 |
| **기업 규모** | 개인사업자, 스타트업(~50명) / 중견(50~500명) / 대기업(500명↑) |
| **거래 유형** | 출입 계약 / 파트너십 계약 / 라이선스 계약 등 |
| **톤** | 친절하고 정중한 말투 |

### 금지어

- **"무료", "공짜", "100%"** — 스팸 필터 트리거
- **"최고", "1등", "업계 유일"** — 과장 표현
- **"지금 당장", "즉시 구매"** — 강압적 표현
- **"법적 책임", "위험", "손해"** — 공포 마케팅
- **경쟁사 직접 언급**

### CTA(Call To Action) 유형 & 필수 포함 항목

- **CTA 유형**: 체험 시작, 데모 예약, 가격 문의
- **필수 포함 항목**: 법적고지(법률 자문이 아님을 명시), 수신 거부 안내

- 수출입 계약 다국어 협상과 내부 결재 시간이 길어지는 경우
- 파트너십/유통 계약이 늘어나며 버전·변경 이력 관리가 어려운 경우

- **가격 문의**: 현재 사용 중인 계약 유형·사용 인원 범위를 알려주시면 맞춤 견적과 요금제 자료를 보내드리겠습니다.
